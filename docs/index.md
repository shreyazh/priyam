# Priyam

Welcome to the priyam documentation! This package provides a comprehensive set of mathematical utilities for Python developers, educators, and researchers.
Modules
1. String Utilities
2. Number Utilities
3. Date & Time Utilities
4. Number Theory
5. Algebra
6. Probability & Statistics
7. LaTeX Support

## Getting Started

### Installation
```bash 
pip install priyam


```python
# Quick Start
from priyam import is_prime, solve_quadratic, mean

# Check if a number is prime
print(is_prime(17))  # True

# Solve a quadratic equation
print(solve_quadratic(1, -3, 2))  # (2.0, 1.0)

# Calculate the mean of a dataset
print(mean([1, 2, 3, 4, 5]))  # 3.0
```
